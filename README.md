This project is a minimal Node.js backend for compiling student report cards.
It supports Teachers, Students, and Administrators with basic token-based authentication.
All data is stored locally in JSON files 


---

Features

Teachers can enter and update Continuous Assessment (CA) and Exam scores.

Automatic calculation of Total Score and Grade.

Students can view their own report card.

Administrators can view and manage all student reports.

RESTful API using Express.js.

Local JSON file storage.



---

Technologies Used

Component Technology

Backend Node.js, Express.js
Data Storage JSON Files (Local)
Authentication Basic Token Header
API Style RESTful JSON
